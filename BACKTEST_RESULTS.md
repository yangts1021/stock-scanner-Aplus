# 台股「主升回檔再啟動」策略回測總結

## 1. 策略原型

針對五條件篩選器（scan_v2.py）抓出的訊號，用歷史資料模擬交易，評估勝率與盈虧比。

### 篩選 5 條件（初版）

| # | 條件 | 參數 |
|---|------|------|
| ① | 長期多頭 | 收盤 > 60MA、60MA 20 日斜率 > 0 |
| ② | 曾有主升段 | 過去 60 日內最大 20 日漲幅 ≥ 30% |
| ③ | 健康回檔 | 自 60 日高點回檔 10~40% |
| ④ | 均線收斂 | 5/10/20MA 三線價差 ≤ 10% |
| ⑤ | 今日啟動 | 漲 ≥ +3% / 站 MA5,MA10 / 高觸 MA20 / 量 ≥ 5MA量×1.5 / K 棒實體 ≥ 10 日均 |

### 資料

- 來源：TWSE/TPEX 官方 API（避開 yfinance 漲停日 NaN 問題）
- 期間：2024-05-14 ~ 2026-05-13（483 個 TW 交易日、351 個 TWO）
- 範圍：上市 1086 + 上櫃 896 = 1982 檔
- SQLite 儲存於 `twstock.db`，支援增量更新

## 4. 關鍵發現

### 4.1 -8% 停損誤殺 30% 好訊號
- 在 v2 的 559 筆 LOSS 中，**30.8%（172 筆）若延長至 60 天會過前高**
- 這些誤殺者「若延長持有」平均報酬 +15.32%
- 停損到突破中位距離 21 天，說明 -8% 停損和 20 天視窗都太緊

### 4.2 啟動 K 棒強度不是有效預測因子
- 漲幅門檻 +3% 改 +5% → EV 從 -1.26% 變 -1.36%（沒效）
- K 棒實體 1.0x 改 1.5x → EV 從 +0.68% 變 +0.10%（變差）
- **動能高低與後續能否過前高無關**

### 4.3 「等回檔」是逆選擇陷阱
- 強勢起飛的訊號根本不回檔，被「等回檔」條件過濾掉
- 真會回檔的本質較弱，更容易跌破前低
- 縮短等待時間（10 天 → 3 天）勝率不變，但進場數變少 → EV 反而更差

### 4.4 R:R 過濾呈「U 型」
| R:R 門檻 | EV(進場) |
|---------|------|
| 1.0 | -0.35% |
| 1.5 | -0.84%（最差）|
| 2.0 | +0.68% |
| **2.5** | **+1.96%（甜蜜點）** |
| 3.0 | -0.51%（過擬合）|

R:R 1.5 反而最差，說明這群「報酬空間中等」的訊號實際表現混雜，不如 R:R 1.0 的雜訊區。
R:R 3.0 失敗印證「條件過嚴 → 樣本太小 → 雜訊噪音放大」。

### 4.5 拉長持有期沒幫助
- HOLD=20、30、40：勝率幾乎不變（23~26%），但 LOSS 隨持有期上升
- 「過前高」要嘛 20 天內發生，要嘛拖久也不會發生
- **20 天是這個型態的天然時間窗口**

### 4.6 大盤環境過濾有效
- 2 年內僅 52% 交易日大盤多頭
- v7 加上大盤多頭過濾後 EV 從 v6 的 -0.91% 改善到 -0.35%
- 系統性風險可以用簡單規則濾除

### 4.7 同檔去重微幅改善
- 30 天 cooldown 讓 EV 從 -1.35% 改善到 -1.26%
- 雜訊有降但不顯著（重複訊號的勝負分布與第一次相當）

## 5. 最終建議策略

### v7 + MIN_RR=2.5（推薦）

**進場規則**
```
1. 篩選器條件全過（scan_vectorized.py，PULLBACK_RANGE=(0.10, 0.20)）
2. 訊號日 TAIEX 收盤 > 60MA 且 60MA 斜率 > 0
3. 計算 R:R = (前高 - 隔日開盤) / (隔日開盤 - 前低)
4. R:R ≥ 2.5 才進場（隔日開盤買入）
5. 同檔 30 天內不重複進場
```

**出場規則**
```
- 過前高（盤中即觸發）→ 以前高價出場（WIN）
- 收盤跌破前低 → 停損出場（LOSS）
- 滿 20 個交易日 → 第 20 天收盤出場（TIMEOUT）
```

**預期表現**
| 指標 | 數值 |
|------|------|
| 訊號頻率 | 每月 2-3 筆 |
| 勝率 | 25% |
| 平均勝幅 | +21% |
| 平均敗幅 | -8% |
| 盈虧比 | 2.65 |
| 單筆 EV（扣費）| **+1.27%** |
| 連敗心理準備 | 連虧 3-4 筆是常態 |

## 6. 風險與限制

| 風險 | 說明 |
|------|------|
| **訊號稀少** | 2 年僅 64 筆，每月 2-3 筆。空窗期長需配合其他策略 |
| **生存者偏差** | DB 只有現存標的，下市股未納入 |
| **市場結構變化** | 回測期 2024-09 ~ 2026-05 涵蓋 1.5 年多頭、半年盤整，未測過長期空頭 |
| **資料量受限** | TPEX 補抓僅至 2024-09-05，可能拖長一些可實證的長期週期 |
| **過擬合風險** | R:R=2.5 是從 1.0/1.5/2.0/2.5/3.0 中挑出，可能在不同時期不再最優 |
| **執行落差** | 模擬假設能以「前高價」出場，實際盤中觸發未必能成交在那個價位（可能要扣 0.5~1% 滑價）|

## 8. 待探索方向

1. **混合策略**：v7 RR=2.5 + 部分倉位 v5（無停損）做避險組合
2. **不同型態**：跌深反彈、突破整理、創新高回測
3. **更精細的大盤條件**：除了 60MA，加 RSI 過熱過濾、波動率閾值
4. **動態 R:R**：依據 prev_high 距離當前價的比例動態調整門檻
5. **加入產業/類股因子**：強勢產業內找 v7 訊號可能勝率更高
6. **更長期資料**：補抓 2023 年甚至 2020 年，驗證跨景氣循環表現
